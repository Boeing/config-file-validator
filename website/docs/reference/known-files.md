# Known Files

The validator recognizes common config files by filename, independent of extension. These mappings are generated from [GitHub Linguist's `languages.yml`](https://github.com/github-linguist/linguist/blob/master/lib/linguist/languages.yml) and updated with each release.

The full list lives in [`known_files_gen.go`](https://github.com/Boeing/config-file-validator/blob/main/pkg/filetype/known_files_gen.go).
